# MERN Daily Counter App

Full MERN stack project:
- Backend: Express + MongoDB + JWT
- Frontend: React + Vite + Tailwind CSS

Run backend:
```
cd server
cp .env.example .env
npm install
npm run dev
```

Run frontend:
```
cd client
cp env.example .env
npm install
npm run dev
```

Default admin credentials (frontend-only):
- username: admin
- password: admin123

API base: http://localhost:5000/api
